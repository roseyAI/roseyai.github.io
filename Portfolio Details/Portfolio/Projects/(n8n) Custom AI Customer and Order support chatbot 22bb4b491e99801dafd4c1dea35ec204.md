# AI Chatbot Integration for E-commerce

## Project Overview

Developed an intelligent customer service chatbot for a Shopify store using n8n workflow automation, integrating Google Gemini AI to provide personalized customer support.

## Technical Implementation

**Tools Used:**

- n8n (Workflow Automation)
- Google Gemini Chat Model
- Shopify API Integration
- Simple Memory Storage

**Architecture:**

- Chat trigger captures customer messages
- AI agent processes inquiries using contextual prompts
- Memory system maintains conversation history
- Shopify integration retrieves order status and product information

## Key Features

- **Contextual Responses**: AI trained on specific website content and service offerings
- **Order Management**: Real-time order status retrieval from Shopify
- **Conversation Memory**: Maintains chat history for personalized interactions
- **Custom Prompting**: Tailored responses based on business context (Mara Baraha store)

## Business Impact

- **Automated Support**: Reduces manual customer service workload
- **24/7 Availability**: Instant responses to customer inquiries
- **Consistent Information**: Ensures accurate product and service details
- **Scalable Solution**: Handles multiple conversations simultaneously

## Technical Skills Demonstrated

- Workflow automation design
- AI model integration and prompt engineering
- API integrations (Shopify, Google Gemini)
- Customer experience optimization
- No-code/low-code development
