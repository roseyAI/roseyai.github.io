Claude Code

Upcoming Stock Scout

A personal stock screener built for growth-focused investors. Discover high-potential emerging stocks across sectors — with real-time data, analyst ratings, and budget-based affordability calculations in PHP.

Visit App

Problem

  • Finding growth stocks is overwhelming — too much noise, too many tools, not enough clarity
  • Most stock screeners are built for institutional investors or require expensive subscriptions
  • Hard to factor in personal budget and local currency (PHP) when evaluating affordability
  • No easy way to filter by values like ESG or screen out high-debt / high-volatility picks

Solution

Built a personal stock screening web app using Claude Code in VS Code. The app pulls real-time stock data, applies custom filters (sector, budget, ESG, risk), and presents stocks in a clean card layout with key metrics — all calculated in Philippine Pesos to make investing decisions more tangible and personal.

Features

  • Sector filtering across Technology & AI, Biotech, FinTech, Consumer, Clean Energy, and Communications
  • Budget calculator in PHP — shows how many shares you can afford at current prices
  • Smart filters to exclude low-growth stocks, high-debt companies, and volatile securities
  • ESG ranking for values-aligned investing
  • Real-time price data with 24-hour change metrics and analyst ratings
  • Growth potential estimates for each screened stock
  • 5-minute data cache for fast, responsive performance

Built With

  • Claude Code (VS Code) — AI-assisted development from scratch
  • HTML, CSS, JavaScript — fully frontend, no frameworks
  • Netlify Functions — serverless API calls for stock data
  • PHP exchange rate integration — live currency conversion
