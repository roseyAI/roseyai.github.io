AI-Powered Interview Answer Grading Workflow

n8n OpenAI Google Sheets AI Agent

Problem

  • Interview candidates struggle to self-assess their answers objectively
  • Manual feedback takes time and isn't scalable
  • Difficult to track progress across multiple practice sessions
  • Hard to know if answers follow proper STAR method structure

Solution

Built an automated interview coaching system that reads practice answers from Google Sheets, uses OpenAI to grade them against STAR methodology, provides detailed feedback, and generates improved versions. The system handles behavioral, hypothetical, and generic questions with customized scoring criteria for each type.

Benefits

  • Instant feedback on interview answers (vs. hours waiting for human review)
  • Consistent, objective grading based on STAR methodology
  • Automated improved answer generation for faster learning
  • Complete practice history tracked in Google Sheets
  • Scores 45 questions in ~5 minutes

How It Works

  1. 1
    Trigger pulls questions and answers from Google Sheets

    Workflow retrieves all interview questions and candidate answers from organized spreadsheet.

  2. 2
    AI Agent (GPT-4.1-mini) grades each answer using STAR criteria

    OpenAI analyzes answers for Situation, Task, Action, and Result components with customized scoring.

  3. 3
    JavaScript code parses AI response and extracts score + feedback

    JSON response processed to extract numerical score and detailed grading notes.

  4. 4
    Results written back to Google Sheets

    Score and feedback automatically populated in designated columns for tracking.

  5. 5
    Second AI Agent generates improved version based on feedback

    System creates enhanced answer incorporating all feedback points for learning.

  6. 6
    Improved answers saved to sheet for candidate review

    Enhanced versions available for practice and comparison with original answers.
